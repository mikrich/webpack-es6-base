# Setup
**Run all commands from root directory**  
**Node Version** 8.1.3  

``` 
$ yarn
```

# Run Dev Mode
**app will be served from localhost:3000**
``` 
$ npm start
```

# Run Prod Mode
**app will be built to /dist directory**  
**app will be served from localhost:3000**
``` 
$ npm run build:serve
```

# Test
``` 
$ npm test
```
